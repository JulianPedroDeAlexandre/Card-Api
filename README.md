# Gerador de Perfil Aleatório

Este projeto é um gerador de perfil de usuário que consome dados reais da API Random User. A aplicação exibe informações como:

- Nome completo
- E-mail
- Data de aniversário
- Endereço
- Telefone
- Senha

Tudo organizado em um card moderno, responsivo e interativo.

## Funcionalidades

- Geração de usuários aleatórios por meio de uma API externa
- Exibição dinâmica das informações no card
- Alteração das informações ao passar o mouse sobre cada opção
- Layout responsivo adaptado para diferentes tamanhos de tela
- Interface simples, limpa e intuitiva

## Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript (Fetch API e manipulação do DOM)

## Objetivo

O projeto foi desenvolvido para praticar:

- Consumo de API externa
- Manipulação de dados e atualização dinâmica do conteúdo
- Estruturação da lógica utilizando funções reutilizáveis
- Desenvolvimento de interfaces responsivas com foco em usabilidade

## Como usar

1. Abra o projeto no navegador
2. Passe o mouse sobre as opções para visualizar diferentes informações do usuário
3. Clique no botão "Generate New User" para carregar um novo perfil
